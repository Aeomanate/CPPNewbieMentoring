# Task
Using only basic arithmetic operators (+, -, *, /) and type conversions, round a given real number to the nearest integer number. 

<details>
<summary>Example 1</summary>
Input: 
  
```
123.456
```
  
Output:
```
123
```
</details>

<details>
<summary>Example 2</summary>
Input: 
  
```
456.789
```
  
Output:
```
457
```
</details>
