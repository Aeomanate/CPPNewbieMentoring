# Task
Using only basic arithmetic operators (+, -, *, /), calc the remainder of a division for two integer numbers.


<details>
<summary>Example 1</summary>
Input: 
  
```
25 8 
```
  
Output:
```
1
```
</details>

<details>
<summary>Example 2</summary>
Input: 
  
```
123 456
```
  
Output:
```
123
```
</details>

<details>
<summary>Example 3</summary>
Input: 
  
```
456 123
```
  
Output:
```
87
```
</details>
