# Task
Using only basic arithmetic operators (+, -, *, /), extract the integer part of a given real number. 

<details>
<summary>Example 1</summary>
Input: 
  
```
123.456
```
  
Output:
```
123
```
</details>

<details>
<summary>Example 2</summary>
Input: 
  
```
112233.445566
```
  
Output:
```
112233
```
</details>
