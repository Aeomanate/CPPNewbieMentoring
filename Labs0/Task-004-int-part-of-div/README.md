# Task
Using only basic arithmetic operators (+, -, *, /), calc the integer part of division for two given numbers. 

<details>
<summary>Example 1</summary>
Input: 
  
```
123 456
```
  
Output:
```
0
```
</details>

<details>
<summary>Example 2</summary>
Input: 
  
```
456 123
```
  
Output:
```
3
```
</details>
